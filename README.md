# ESP32 IoT Firebase Flutter System

## Overview
This project is developed as a learning project to explore IoT system design using ESP32 and Flutter.
The system focuses on device monitoring and remote control via a mobile application.

## System Components
- ESP32: Handles sensor data acquisition and device control
- Firebase Realtime Database: Cloud-based data synchronization
- Flutter App: Mobile application for user interaction

## Features
- Basic sensor data monitoring
- Remote control of ESP32 devices
- WiFi-based communication
- Firebase integration

## Learning Objectives
- Understand ESP32 WiFi communication
- Learn Flutter application development
- Practice cloud integration using Firebase
- Experience basic IoT system architecture
